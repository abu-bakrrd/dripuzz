# Добавление сайта в Google и Яндекс

Пошаговая инструкция для продвижения вашего магазина в поисковиках.

---

## Часть 1: Добавление в Google

### Шаг 1. Откройте Google Search Console

1. Перейдите: https://search.google.com/search-console
2. Войдите в аккаунт Google

### Шаг 2. Добавьте сайт

1. Нажмите **"Добавить ресурс"** (слева вверху)
2. Выберите **"Префикс URL"** (правая колонка)
3. Введите полный адрес сайта: `https://ваш-домен.com`
4. Нажмите **"Продолжить"**

### Шаг 3. Подтвердите владение

**Самый простой способ — через HTML-тег:**

1. Google покажет код вида:
   ```html
   <meta name="google-site-verification" content="ABC123..." />
   ```
2. Этот тег нужно добавить в `<head>` вашего сайта
3. Откройте файл `client/index.html` и добавьте тег внутри `<head>`:
   ```html
   <head>
     <meta name="google-site-verification" content="ВАШ_КОД_ЗДЕСЬ" />
     ...
   </head>
   ```
4. Пересоберите и обновите сайт на VPS:
   ```bash
   npm run build
   sudo systemctl restart shop-app
   ```
5. Вернитесь в Google Search Console и нажмите **"Подтвердить"**

### Шаг 4. Отправьте карту сайта (Sitemap)

1. В меню слева выберите **"Файлы Sitemap"**
2. В поле "Добавить новый файл Sitemap" введите:
   ```
   sitemap.xml
   ```
3. Нажмите **"Отправить"**
4. Статус должен стать **"Успешно"**

**Если ошибка "Недопустимый файл":**
- Проверьте что сайт работает: откройте `https://ваш-домен.com/sitemap.xml` в браузере
- Должен отобразиться XML с вашими товарами
- Если страница не открывается — перезапустите приложение на VPS

## Часть 2: Добавление в Яндекс

### Шаг 1. Откройте Яндекс.Вебмастер

1. Перейдите: https://webmaster.yandex.ru
2. Войдите в Яндекс-аккаунт

### Шаг 2. Добавьте сайт

1. Нажмите **"+ Добавить сайт"**
2. Введите адрес: `https://ваш-домен.com`
3. Нажмите **"Добавить"**

### Шаг 3. Подтвердите владение

1. Выберите способ **"Мета-тег"**
2. Скопируйте код:
   ```html
   <meta name="yandex-verification" content="ABC123" />
   ```
3. Добавьте в `client/index.html` рядом с Google-тегом
4. Обновите сайт на VPS
5. Нажмите **"Проверить"**

### Шаг 4. Отправьте Sitemap

1. Перейдите в раздел **"Индексирование" → "Файлы Sitemap"**
2. Нажмите **"Добавить"**
3. Введите: `https://ваш-домен.com/sitemap.xml`
4. Нажмите **"Добавить"**

## Проверка работы Sitemap

### На вашем сайте

Откройте в браузере:
```
https://ваш-домен.com/sitemap.xml
```

Должен отобразиться XML примерно такого вида:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://ваш-домен.com/</loc>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://ваш-домен.com/product/abc123</loc>
    ...
  </url>
</urlset>
```

### Через командную строку (на VPS)

```bash
curl -I https://ваш-домен.com/sitemap.xml
```

Должен вернуть:
```
HTTP/2 200
content-type: application/xml; charset=utf-8
```

### Если sitemap не работает

1. Проверьте логи:
   ```bash
   sudo journalctl -u shop-app -n 50 --no-pager
   ```

2. Перезапустите приложение:
   ```bash
   sudo systemctl restart shop-app
   ```

3. Проверьте Nginx:
   ```bash
   sudo nginx -t
   sudo systemctl reload nginx
   ```

## Что ещё сделать для SEO

### Описания товаров

Напишите уникальные описания для каждого товара (150-300 слов).

**Плохо:** `Платье. Хорошее качество.`

**Хорошо:** `Элегантное женское платье из натурального хлопка. Мягкая ткань обеспечивает комфорт. Подходит для повседневной носки. Размеры S-XL.`

### Изображения

- Минимум 800x800 пикселей
- Сжимайте перед загрузкой: https://tinypng.com
- Добавляйте несколько фото с разных ракурсов

Создайте страницы магазина:

- Instagram
- Facebook

Добавьте ссылку на сайт в описание профилей.
