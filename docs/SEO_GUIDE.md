# SEO Руководство

Это руководство поможет вам продвинуть ваш интернет-магазин в поисковых системах.

## Что уже сделано (техническая часть)

### 1. Мета-теги
- Title и description автоматически генерируются для каждой страницы
- Open Graph теги для красивого отображения в социальных сетях
- Canonical URL для избежания дублирования контента

### 2. Структурированные данные (Schema.org)
- **Product Schema** - для каждого товара (название, цена, наличие)
- **Organization Schema** - информация о магазине
- **BreadcrumbList Schema** - навигационные хлебные крошки

### 3. Sitemap.xml и Robots.txt
- `/sitemap.xml` - карта сайта со всеми товарами и категориями
- `/robots.txt` - правила индексации для поисковых роботов

## Что нужно сделать вам

### Шаг 1: Зарегистрируйте сайт в Google Search Console

1. Перейдите на [Google Search Console](https://search.google.com/search-console)
2. Нажмите "Добавить ресурс"
3. Выберите "Домен" или "Префикс URL" и введите адрес вашего сайта
4. Подтвердите владение через DNS или HTML-файл
5. После подтверждения отправьте sitemap:
   - Перейдите в раздел "Карты сайта"
   - Введите `sitemap.xml` и нажмите "Отправить"

### Шаг 2: Зарегистрируйтесь в Яндекс.Вебмастер (для СНГ)

1. Перейдите на [Яндекс.Вебмастер](https://webmaster.yandex.ru)
2. Добавьте сайт и подтвердите владение
3. Отправьте sitemap.xml

### Шаг 3: Качественные описания товаров

**Важно!** Уникальные описания - ключ к хорошему ранжированию.

Для каждого товара напишите:
- **Заголовок**: 50-60 символов, включите ключевые слова
- **Описание**: 150-300 слов, уникальный текст
- Укажите характеристики, преимущества, способы использования

**Плохой пример:**
> Платье женское. Хорошее качество.

**Хороший пример:**
> Элегантное женское платье из натурального хлопка. Идеально подходит для повседневной носки и особых случаев. Мягкая ткань обеспечивает комфорт в течение всего дня. Доступно в размерах S-XL.

### Шаг 4: Качественные изображения

- Используйте изображения высокого качества (минимум 800x800 пикселей)
- Сжимайте изображения перед загрузкой (используйте [TinyPNG](https://tinypng.com))
- Добавляйте несколько фото товара с разных ракурсов

### Шаг 5: Получайте отзывы

Отзывы улучшают ранжирование:
- Просите клиентов оставлять отзывы
- Отвечайте на отзывы
- Добавьте возможность оставлять отзывы на сайте (если ещё нет)

### Шаг 6: Ссылки с других сайтов (Backlinks)

Качественные ссылки на ваш сайт повышают авторитет:
- Разместите магазин в каталогах (2ГИС, Яндекс.Карты)
- Сотрудничайте с блогерами
- Создайте страницы в социальных сетях с ссылкой на сайт
- Публикуйте гостевые статьи

### Шаг 7: Социальные сети

- Создайте аккаунты в Instagram, Telegram, Facebook
- Добавьте ссылки на сайт во всех профилях
- Регулярно публикуйте контент с товарами

### Шаг 8: Скорость загрузки

Проверьте скорость сайта:
1. Перейдите на [PageSpeed Insights](https://pagespeed.web.dev)
2. Введите URL вашего сайта
3. Следуйте рекомендациям по улучшению

---

## Ожидаемые сроки

SEO - это долгосрочная стратегия:
- **1-2 недели**: Индексация сайта поисковиками
- **1-3 месяца**: Первые результаты в поиске
- **6-12 месяцев**: Стабильные позиции

## Дополнительные рекомендации

### Локальное SEO (для физических магазинов)
1. Создайте профиль в Google Мой Бизнес
2. Добавьте магазин в Яндекс.Справочник
3. Укажите точный адрес, часы работы, фото

### Контент-маркетинг
- Ведите блог с полезными статьями
- Публикуйте обзоры товаров
- Создавайте гайды по выбору продукции
